# Ainekio the Robot Familiar

Ainekio is a hardware and software project for building a small robot companion powered by MetaHuman OS. The robot uses a physical body, onboard sensors, and a network connection so MetaHuman OS can see, listen, speak, move, and interact through a physical chassis.

This site is the public field notebook for the project. It will collect build notes, design decisions, safety boundaries, experiments, and progress updates as the robot takes shape.

## Current Starting Point

- The project repo is now `Greg-Aster/Ainekio-bot`.
- The public domain is `ainek.io`.
- The current test body is based on the Sesame robot.
- MetaHuman OS remains the companion intelligence layer; hardware, firmware, cloud relay, and robot behavior stay explicit and separate.

## Project Shape

Ainekio is not meant to expose raw servo control or unsafe motion instructions. Robot behavior should be expressed through semantic commands, checked by safety gates, and kept understandable enough to debug.

Early work will focus on a small set of practical questions: what the body can safely do, what sensors are available, how the network bridge should behave, and which interactions actually feel useful.
